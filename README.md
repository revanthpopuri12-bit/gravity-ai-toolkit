# Gravity AI Toolkit

This project provides AI-driven tools for analysing gravitational-wave and extreme astrophysical signals.

## Features (Initial)
- Signal preprocessing (filtering, whitening, normalisation)
- Visual diagnostics for waveform analysis
- Modular pipeline for research use

## Goal
To build a research-grade toolkit that helps scientists analyse and interpret extreme gravitational phenomena such as black hole mergers.
